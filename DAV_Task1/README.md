
# Task 4 - Student Marks Prediction

## Objective
Build a machine learning model to predict student exam scores based on study hours and visualize the relationship using a line of best fit.

## Activities Performed
- Loaded and analyzed the student scores dataset
- Prepared input and output features
- Trained a Linear Regression model
- Predicted student exam scores
- Evaluated model performance using R² Score
- Visualized the line of best fit

## Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn
- Google Colab

## Machine Learning Concepts Used
- Supervised Learning
- Linear Regression
- Model Training & Prediction
- Data Visualization
- Performance Evaluation
