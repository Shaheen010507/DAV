# Sales Prediction using Machine Learning

## Project Overview

This project predicts product sales based on advertising expenditure using Machine Learning. Historical advertising data is used to train a Linear Regression model that estimates future sales. The project demonstrates the complete data mining workflow, including data preprocessing, visualization, model training, evaluation, and prediction.

## Objectives

* Analyze advertising data.
* Understand the relationship between marketing expenditure and sales.
* Build a Machine Learning model for sales prediction.
* Evaluate the model's performance.
* Predict sales for new advertising budgets.

## Dataset

A synthetic dataset containing 300 records was created for this project.

### Dataset Attributes

* **TV** – Advertising budget spent on TV.
* **Radio** – Advertising budget spent on Radio.
* **Newspaper** – Advertising budget spent on Newspaper.
* **Sales** – Product sales corresponding to the advertising budget.

## Tools and Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn

## Steps Performed

1. Imported the required Python libraries.
2. Loaded the sales dataset (`sales_300rows.csv`).
3. Explored and verified the dataset.
4. Selected TV, Radio, and Newspaper as input features.
5. Selected Sales as the target variable.
6. Split the dataset into training and testing sets.
7. Trained a Linear Regression model.
8. Predicted sales using the trained model.
9. Evaluated the model using Mean Squared Error (MSE).
10. Tested the model with new advertising values to predict future sales.

## Machine Learning Algorithm

**Linear Regression**

Linear Regression is a supervised Machine Learning algorithm that predicts continuous numerical values by identifying the relationship between input features and the target variable.

## Model Evaluation

The model is evaluated using:

* Mean Squared Error (MSE)
* Prediction accuracy through comparison of actual and predicted values
